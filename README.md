# maintenance-app      
L’ Application de Gestion de Maintenance
Ce projet est une application Java conçue pour aider un service de maintenance à s'organiser au quotidien. Elle permet de suivre les interventions techniques, de gérer les équipes de techniciens et de répertorier les bâtiments où nous intervenons.
Ce qu’on peut faire avec l'appli
Gérer les techniciens : Ajouter de nouveaux membres, modifier leurs qualifications ou voir s'ils sont disponibles.
Gérer les bâtiments : Créer une liste des lieux d'intervention avec leur adresse.
Planifier les interventions : Programmer une réparation ou un contrôle en choisissant le technicien et le lieu, puis suivre l'avancement (Planifiée, En cours ou Terminée).
Consulter l'historique : Filtrer les interventions pour voir le planning d'un technicien ou l'historique d'un bâtiment.
